# FinFlow — Инструкция по деплою на GitHub Pages

## 🚀 За 5 шагов

1. **Создай репозиторий на GitHub**
   - Зайди на [github.com](https://github.com) → кнопка **"New"**
   - Название: `finflow` (или любое другое)
   - Видимость: **Public** ✅ (для бесплатного GitHub Pages)
   - Нажми **"Create repository"**

2. **Загрузи файл**
   - На странице репозитория нажми **"Add file" → "Upload files"**
   - Перетащи файл `index.html`
   - Внизу нажми **"Commit changes"**

3. **Включи GitHub Pages**
   - Перейди в **Settings** (вкладка в верхнем меню репозитория)
   - В боковом меню найди **Pages**
   - В разделе **"Branch"** выбери `main` и папку `/ (root)`
   - Нажми **Save**

4. **Подожди 1–2 минуты**
   - GitHub покажет ссылку вида:
     `https://ВАШ_ЛОГИН.github.io/finflow/`

5. **Открывай на любом устройстве!**
   - Ссылка работает на телефоне, планшете, ПК
   - Можно добавить на главный экран iPhone/Android (работает как приложение)

---

## 📱 Как добавить на главный экран

### iPhone (Safari)
1. Открой сайт в Safari
2. Нажми кнопку **Поделиться** (квадрат со стрелкой)
3. Выбери **"На экран «Домой»"**

### Android (Chrome)
1. Открой сайт в Chrome
2. Нажми **⋮** (три точки) в правом верхнем углу
3. Выбери **"Добавить на главный экран"**
